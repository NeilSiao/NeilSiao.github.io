Install PHPUnit

前置安裝:
- Composer (PHP套件管理工具)
- PHP

[(PHPUnit) [https://phpunit.de/]]，PHP的測試套件，用來撰寫程式測試用。根據官網的介紹，安裝步驟如下:

-  composer require --dev phpunit ^9
-  ./vendor/bin/phpunit --version  ## Windows 需要改寫 "./vendor/bin/phpunit" --version

PHPUnit官網上的範例程式碼

Running PHPUnit

![PHPUnit 執行畫面](/img/phpunitCmd.jpg)
-php phpunit
